System Requirements

Minimum system requirements to use WISPGATE:

  1. Operating System: Ubuntu 18.04+, Debian 8.4+, Centos (RedHat, Windows Core Server 12r2+

  2. CPU: 64-bit i3+ (x64) Processor

  3. Memory: 8 GB RAM

  4. Hard Disk: 200 GB available disk space
